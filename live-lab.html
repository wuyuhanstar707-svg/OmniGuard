USB-C? Many Type-C cables are power-only—the PC will not see the Arduino. Use a data cable, try USB-A + adapter, install board drivers (CH340 / CP210x), and close Arduino Serial Monitor. Full guide →

Arduino sketch tips (why the stream might look empty)
  • The box below shows only data from your board—not errors from the website.
  • Match baud: Serial.begin(115200) here = 115200 in the menu above (or both 9600).
  • The chart needs a number per line, or JSON with a numeric field — JSON help.

Paste this sketch in Arduino IDE → Upload:

void setup() {
  Serial.begin(115200);
}
void loop() {
  Serial.println(analogRead(A0));
  delay(100);
}
